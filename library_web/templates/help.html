Help & Support

Welcome to the Library Management System! Here, you can find helpful information on how to use the system.

How to Search for Books

To search for a book in our library, simply use the search bar at the top of the home page. You can search by book title, author, or ISBN number.

How to Borrow Books

Once you’ve found the book you’re looking for, click on it for more details. If it’s available, you can borrow it by clicking the "Borrow" button. Remember, you need to be logged in to borrow books.

How to Return Books

When you are done reading a book, you can return it via your user dashboard. Go to "My Books" and click "Return" next to the book you'd like to return.

How to Contact Admin

If you have any questions or issues, you can contact the admin through the "Contact Us" page. You can send a message directly to the admin using the contact form provided.

How to Update Your Profile

To update your profile, go to your user dashboard and click on "Profile Settings." Here, you can update your username, email, and password.

If you need further assistance, please contact support at support@library.com
